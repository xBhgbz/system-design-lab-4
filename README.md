## Запуск

```bash
docker-compose up --build
```

Сервис будет доступен на `http://localhost:8080`
MongoDB будет доступна на `mongodb://localhost:27017`

## Тесты

```
bash tests.sh
```

### Инициализация БД

При первом запуске `docker-compose` автоматически:
1. Создает базу данных `hotel_booking`
2. Создает коллекции с валидацией схем
3. Вставляет тестовые данные (10 пользователей, 10 отелей, 10 бронирований)
